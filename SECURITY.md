At Codex Arcanum, we take the security of our project and the protection of our community seriously. We appreciate the efforts of security researchers and the wider community in helping us identify and address potential vulnerabilities.

## Reporting Security Vulnerabilities

If you discover a security vulnerability within the Codex Arcanum project, please follow these steps to report it securely:

1. Send an email to our dedicated security email address: security@codexarcanum.ai.
2. Include a detailed description of the vulnerability, along with any necessary steps to reproduce it.
3. Provide your contact information so that our security team can follow up with you if needed.
4. Allow us a reasonable amount of time to investigate and address the vulnerability before disclosing it publicly.

Please do not report security vulnerabilities through public channels such as GitHub issues, Telegram groups, or social media platforms.

## Scope

Our security policy applies to the following components of the Codex Arcanum project:

- Smart contracts
- Web applications
- Backend infrastructure
- Any other critical components identified by our team

## Response Process

Upon receiving a security vulnerability report, our team will take the following steps:

1. Acknowledge receipt of the report within 48 hours.
2. Assess the severity and impact of the vulnerability.
3. Develop and test a fix for the vulnerability.
4. Deploy the fix to our production environment.
5. Notify the reporter once the vulnerability has been addressed.
6. Publicly disclose the vulnerability and credit the reporter (if desired) after an agreed-upon timeframe.

## Coding and Final Report

Please note that the Codex Arcanum project is currently under active development, and we are in the process of coding the project. The final report, along with the main code, will be released upon completion of the development phase.

We appreciate your patience and understanding during this time. Rest assured that security remains a top priority throughout the development process, and we will continue to follow best practices and conduct thorough security audits before the final release.

## Contact

If you have any questions or concerns regarding the security of the Codex Arcanum project, please contact our security team at security@codexarcanum.com.

Thank you for helping us keep the Codex Arcanum project secure!
